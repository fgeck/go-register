# go-register
Template repo for user registration and signin based on goth stack

## Tools used

* [echo](https://echo.labstack.com/)
* [sqlc](https://sqlc.dev/)
* [templ](https://github.com/a-h/templ)
* [tailwindcss](https://tailwindcss.com/)
* [daisyui](https://daisyui.com/)
